# Stock Market Analysis & Predictive Modeling 📈

## Project Overview
This project analyzes historical stock data for **Reliance Industries (RELIANCE.NS)** to identify volatility patterns and forecast future price movements using statistical models.

## Key Features
* **Data Extraction:** Automated data fetching using the `yfinance` API.
* **Risk Metrics:** Calculation of Annualized Volatility and Sharpe Ratio.
* **Predictive Modeling:** Comparison between Linear Regression and **ARIMA** time-series forecasting.
* **Visualizations:** Clean data plotting using Matplotlib and Seaborn.

## Results
* **Current Model RMSE:** 148.06
* **Sharpe Ratio:** 0.60 (Indicates the risk-adjusted return profile)

## How to Run
1. Install requirements: `pip install yfinance pandas statsmodels matplotlib seaborn`
2. Open `Stock_market_project.ipynb` in Jupyter Notebook or VS Code.
